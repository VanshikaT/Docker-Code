This file contains a list of Docker CLI commands.

### Check Docker Version

```bash
docker --version
```

NOTE: hello-world used below is a docker image.

### Run a container

```bash
docker run hello-world
```

### Pull a docker image

```bash
docker pull hello-world
```

### For checking only running containers

```bash
docker ps
```

### For checking all the containers

```bash
docker ps -a
```

### For fetching available images

```bash
docker images
```
